Introduction

The FLAGLET code provides high performance routines for fast wavelet analysis of signals on the ball. It is primarily intended to work with the SSHT code built on the MW sampling (ArXiv | DOI) to perform exact spherical harmonic transform on the sphere. The resulting wavelet transform is also theoretically exact, i.e. it captures all the information of band-limited signals in the minimal number of samples. FLAGLET was also extended to support the HEALPIX sampling scheme, in which case the transforms are not theoretically exact but achieve good numerical accuracy.

This page outlines the main features of FLAGLET, installation details as well as the core functionalties and interfaces. References, version, and license information then follows. FLAGLET requires the SSHT and FFTW libraries. The IO FITS features require CFITSIO. To support HEALPIX, a valid installation of its Fortran implementation must be provided.


Wavelet transform on the sphere

In FLAGLET, the wavelet kernels are constructed through exact tilling of Fourier-Laguerre space, a conjunction of the harmonic (for the sphere) and the Laguerre multipoles (for the radial half-line). This 2D harmonic space is tiled into wavelet kernels ("flaglets") which are localised in both real and frequency spaces, as shown on the image below for a particular set of wavelet parameters (tiling of the Fourier-Laguerre space on the left, some flaglet kernels on the right).

      

With an exact spherical harmonic transform as the one provided in SSHT and the exact spherical Laguerre transform (see related publications), the wavelet transform implemented in FLAGLET is theoretically exact in both pixel and harmonic spaces. In other words one can decompose a band-limited signal in a set of flaglet maps that exactly capture all the information, and reconstruct the initial signal at floating-point precision. Due to the nature of the tiling in Fourier-Laguerre space, the individual flaglets have different band-limits and hence can be reconstructed at different resolutions, i.e. on the minimal number of samples. Due to the exactness of all transforms, these two approaches are equivalent and capture all the information contained in the initial band-limited map.


Installation and documentation

The core functionalities of FLAGLET are written in C and self-documenting. The C library requires FLAG, S2LET, SSHT and FFTW. Interfaces are provided for all high-level routines in Matlab. Several examples, tests and demos are provided fo the C library and the Matlab interfaces.
